# Meta

논문 관리용 메타 문서를 모아두는 폴더

## Structure

```text
99_Meta/
├── meta_guide.md
├── paper_index.md
├── reading_notes/
└── experiment_ideas/
```

## Files

| Path | Purpose |
|---|---|
| `paper_index.md` | 전체 논문 목록과 읽기 노트 링크 관리 |
| `reading_notes/` | 논문별 개별 읽기 노트 |
| `experiment_ideas/` | 실험 아이디어별 개별 메모 |

## Workflow

1. PDF를 주제 폴더에 추가한다.
2. `paper_index.md`에 새 논문 행을 추가한다.
3. `reading_notes/`에 `ID_ShortName.md` 형식의 노트 파일을 만든다.
4. 논문을 읽다가 실험 아이디어가 생기면 `experiment_ideas/`에 별도 파일로 정리한다.
